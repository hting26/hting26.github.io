瑪莉愛狗協會網站
使用 Vue CLI 及 Node.js 開發的網站，搭配 MongoDB 資料庫。
是以替原本較舊的網站改版為出發點，可讓訪客登入會員、選購捐贈物資、送出訂單、申請領養狗狗等。

  • 新增、編輯商品，查看訂單。
  • 新增、編輯領養犬，查看、刪除申請。
